Weight

It makes vertices of the patches-selected object angular by setting up weight. Weight can be set from -1.0 to 1.0.

Weight is only available for 'Spline type 1' or 'Spline type 2' patches. It makes no change for 'Catmull-Clark' patches.

When you drag a mouse on the selected face, you can see the change of the shape in real time.
