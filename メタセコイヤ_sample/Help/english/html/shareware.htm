About shareware version

A shareware version of Metasequoia Ver3.0 is provided as an upper edition of MetasequoiaLE R3.0. Some functions are not available in MetasequoiaLE. Please upgrade to shareware version when you want to use them.

If you use shareware version of Metasequoia, you must send money to purchase a license. Please see details how to purchase a license in the official web site.

Functions for shareware only

  • Three orthogonal views
  • A hierarchy for an object, a local coordinate
  • Vertex color
  • Weight for a patch
  • Fractal landscape
  • File import (3DS and so on)
  • File export (LWO, 3DS, VRML, OBJ and so on)
  • Plug-ins (Text, Boolean and so on)
  • Plug-in SDK, Python script
  • etc.
